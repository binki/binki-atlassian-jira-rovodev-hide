Remove the “Rovo Dev” section that Jira shows inline in an issue view.

This increases the height of content in a page, often requiring you to scroll before seeing actual content in the issue.

![“Rovo Dev” section shown between “Linked work items” and “Activity” in a Jira issue.](https://i.imgur.com/Nigcrtr.png)
